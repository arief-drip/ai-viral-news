2/10 APA ITU?

Prompt Injection

Celah keamanan spesifik AI — celah yang memungkinkan peretas menimpa instruksi asli sistem dengan perintah berbahaya.

Mirip social engineering, tapi untuk AI. Penyerang menyisipkan instruksi tersembunyi dalam input yang tampak normal.

3/10 CELAH KLASIK

"Ignore All Previous Instructions"

Serangan paling populer: user menyuruh AI mengabaikan semua instruksi sistem dan mengikuti perintah berbahaya.

Contoh nyata:

  • Membocorkan data pribadi dari database
  • Mengubah perilaku AI untuk merugikan pengguna
  • Mengekstrak informasi rahasia perusahaan
AI VIRAL NEWS
4/10 SOLUSI

Instruction Hierarchy

OpenAI mengembangkan teknik "Instruction Hierarchy" — hierarki instruksi yang memberi prioritas pada perintah sistem.

Cara kerja:

  • System message = prioritas TERTINGGI
  • User prompt = prioritas lebih rendah
  • Konflik? Sistem menang
AI VIRAL NEWS
5/10 IMPLEMENTASI

GPT-4o Mini Jadi Pertama

Model pertama yang mendapat lapisan keamanan ini adalah GPT-4o Mini — model ringan dan murah dari OpenAI.

"Ini mengajarkan model untuk benar-benar mengikuti pesan sistem developer," kata Olivier Godement, Head of API Platform OpenAI.

Ketika ditanya apakah ini akan menghentikan serangan "ignore all instructions"? Jawabannya: "Tepat sekali."

6/10 CARA KERJA

System Message > User Prompt

LLM saat ini kesulitan membedakan instruksi sistem vs perintah user. Instruction Hierarchy menyelesaikan ini dengan:

1

Evaluasi konflik antar instruksi

2

Prioritas berdasarkan asal instruksi

3

Tolak perintah yang bertentangan dengan sistem

7/10 MASA DEPAN

Menuju AI Agent yang Aman

Keamanan ini adalah fondasi untuk masa depan AI Agent otonom yang bisa menjalankan tugas kompleks atas nama kita.

Bayangkan AI Agent yang bisa:

  • Mengelola email dan kalender
  • Bertransaksi online
  • Mengakses database perusahaan

Semua butuh proteksi prompt injection yang kuat.

AI VIRAL NEWS
8/10 DAMPAK

Membangun Kembali Kepercayaan

Kepercayaan publik terhadap AI sempat terguncang oleh rentetan serangan prompt injection.

Dengan Instruction Hierarchy, OpenAI menunjukkan komitmen pada keamanan sebelum meluncurkan agent berskala besar.

Langkah ini krusial untuk meyakinkan pengguna bahwa AI bisa dipercaya mengelola data dan tugas pribadi.

AI VIRAL NEWS
9/10 RENCANA KE DEPAN

Guardrails yang Lebih Kompleks

OpenAI membayangkan sistem keamanan berlapis seperti web browser modern — bukan satu fitur, tapi ekosistem perlindungan.

"Kami membayangkan berbagai jenis guardrail yang lebih kompleks di masa depan," kata tim peneliti OpenAI.

Ini baru awal. Keamanan AI akan terus berkembang seiring adopsi agent yang semakin luas.
